***

### 功能

- v1.0.4
  1. Explore Github
  2. 搜索repository、user；
  3. star/watch/fork repository；
  4. follow/unfollow user;
  5. 查看notification和issue;
  6. 查看用户的 starred/watched repositories；
  7. 查看用户的 event;
  8. 查看用户的 followers/following user;
